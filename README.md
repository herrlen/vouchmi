# TrusCart

**Community Commerce Platform**

Nutzer erstellen Communities, teilen Produktlinks und entdecken Deals — kostenlos. Marken zahlen für Sichtbarkeit.

## Quick Start

### App (React Native)
```bash
cd app
npm install
npx expo start
```

### Backend (Laravel)
```bash
cd backend
composer install
cp .env.example .env
php artisan key:generate
php artisan migrate
```

## Struktur

| Ordner | Was | Tech |
|--------|-----|------|
| `app/` | Mobile App (iOS + Android) | React Native, Expo, TypeScript |
| `backend/` | API Backend | Laravel 11, PHP, MySQL |
